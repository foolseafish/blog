# 冒泡排序
## 描述
- 冒泡排序，原地排序，稳定排序。
- 时间复杂度 O(n2)
## 介绍
类比电子竞技游戏的小组赛(规定每个小组比一次)，一共x位参赛队伍，抓一个挨个打过去(打赢了第一个找第二个打)，如果他全胜则就是第一名，剩下的人竞争第二名，第三名...。
## 思路
1. 一共x位参赛队伍，抓一个挨个打过去（外循环i）
2. 打赢了第一个找第二个打(内循环j)
3. 剩下的人竞争第二名，第三名（外循环i--,因为只能竞争第2第3了）

## 实现
```javascript
const bubbleSort = (arr)=>{
    const len = arr.length;
    for(let i = len - 1; i > 0; i--){
        for(let j= 0; j<=i; j++){
            if (arr[j]>arr[j + 1]) {
                const temp = arr[j+1];
                arr[j+1] = arr[j];   
                arr[j] = temp;
            } 
        }
    }
}
```
